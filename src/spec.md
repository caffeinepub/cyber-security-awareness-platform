**Goal:** Correct the Hindi spelling of "Vachellia Psychocare" from वैशेलिया to वैचेलिया throughout the application.

**Planned changes:**
- Update Header component logo text to display 'वैचेलिया साइकोकेयर'
- Update HomePage hero section heading to show 'वैचेलिया साइकोकेयर'
- Update all translations in utils/translations.ts to use 'वैचेलिया साइकोकेयर'
- Remove all instances of the old spelling 'वैशेलिया' from the codebase

**User-visible outcome:** Users will see the corrected Hindi spelling "वैचेलिया साइकोकेयर" displayed consistently throughout the application in the header and homepage.
